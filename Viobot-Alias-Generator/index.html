Alias setup name & permissions

An alias is a saved custom command. You give it a name and decide who can run it. Later you paste the finished command into Discord once, and from then on people just type !name.

Public = saved for the whole server. Private = a personal shortcut only you can use.
Mods only = just your staff team. Everyone = any member.

Inputs optional

Extra words someone can type after the command — like a user to tag or a reason. Name them here, then drop them into your message with Insert ▾. (Skip this if your alias is the same every time.)

User = someone they @mention or paste an ID for.   Text = any typed words.   rest = grab everything from that point on (for a multi-word reason — keep it last).

Example: add an input named user (User). Now Insert ▾ → Tag user drops them into your text, so running !welcome @Bob greets Bob by name.

Messages what the bot sends

What Viobot actually posts when the alias runs. Each block is one Discord message — give it plain text, a Card (a coloured info box), or both. Add more than one block to send several messages in a row.

Live preview how it'll look

Exactly how Discord will show your message. To see your inputs/variables fill in, type a sample below as if someone ran the command.

Mentions are shown for preview only — Viobot sends with pings suppressed, so running the alias won't actually notify anyone. Images/thumbnails only appear if the URL is valid.

Show text only when…

      
A condition shows text only in certain cases — Viobot checks it each time the alias runs. Example: tag the user only if they actually gave one. "Show this" appears when the condition is true; "Otherwise" shows when it isn't.
